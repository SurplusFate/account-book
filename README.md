# 账号本子 · Account Book

网页版加密账号密码管理工具。所有数据在浏览器本地端到端加密存储，通过 WebDAV 协议连接坚果云进行云同步。无后端服务器、无账号注册、无会员体系——只需一个主密码。

## 特性

- **端到端加密**：使用浏览器原生 Web Crypto API（PBKDF2 派生密钥 + AES-GCM 加密），主密码从不离开设备，坚果云仅存储密文
- **坚果云 WebDAV 同步**：基于坚果云第三方应用专用密码，支持手动 / 自动同步，多设备间账号级合并
- **账号管理**：分类、收藏、置顶、关键词搜索（含拼音首字母）、密码生成器、一键复制
- **安全选项**：自动锁定、剪贴板自动清除、加密备份导入导出、主密码修改
- **纯前端**：React + TypeScript + Vite，零服务器成本，可部署到任意静态托管

## 快速开始

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建生产版本
pnpm build

# 预览生产构建
pnpm preview
```

## 配置坚果云同步

1. 登录 [坚果云](https://www.jianguoyun.com/) → 账户信息 → 安全选项 → 第三方应用管理
2. 点击「添加应用」，生成一个**应用专用密码**
3. 在本应用「同步」页面填写：
   - 服务器地址：`https://dav.jianguoyun.com/dav/`（已预填）
   - 坚果云账号：你的登录邮箱 / 手机号
   - 应用专用密码：上一步生成的密码
   - 同步目录：如 `account-book`
4. 点击「测试连接」→ 保存配置 → 「立即同步」

|----|------|
| 框架 | React 18 + TypeScript |
| 构建 | Vite |
| 样式 | TailwindCSS 3 |

| 存储 | IndexedDB（idb） |
| 同步 | 自实现 WebDAV 客户端 |

## 安全说明

- 主密码通过 PBKDF2-SHA256（150000 次迭代）派生 256 位密钥
- 所有账号数据使用 AES-GCM 加密，每次保存生成随机 IV
- 云端只存储加密产物，即使坚果云账号泄露，没有主密码也无法解密
- **主密码无法找回**，请妥善保管；建议导出加密备份
